## Overview

The Record Detail page provides comprehensive views of individual dataset records, combining structured data visualization with notebook-style documentation capabilities.

**Two views**:

- **Properties Tab**: Structured field values with evidence and sources
- **Notebook Tab**: Rich text editor for creating documentation with AI assistance

## Properties View

**Field data display**: Column names with data type indicators, processed data with formatting, AI reasoning behind generated values, source indicators with favicons.

**Interactive features**: Search fields by name or content, automatic formatting, click sources for detailed attribution, processing status indicators.

**AI evidence**: Detailed reasoning, source links with previews, confidence indicators, processing history timeline.

**Rich editor** (Tiptap): Rich formatting (bold, italic, code), structured content (headings, lists, blockquotes), tables with full editing, links, task lists, syntax-highlighted code blocks, images, horizontal rules.

**Slash commands**: Type `/` for paragraphs, headings (`/heading1-3`), lists (`/bulletlist`, `/orderedlist`), tasks (`/taskitem`), blockquotes, code blocks, tables, horizontal rules, AI generation (`/ai`).

**AI integration**: Context awareness (understands current record), agent selection, real-time streaming, tool integration for research, generates contextual analysis and insights.

## AI Content Generation

**Capabilities**: Analysis reports from record data, research summaries with external information, data insights and patterns, action plans and recommendations, comparison tables with other records.

**Tool integration**: Web search for current information, API calls to external services, data validation from multiple sources, enrichment with missing context.

## Use Cases

**Research**: Company analysis with market position research, contact profiling with interaction history, meeting summaries, relationship intelligence.

**Data quality**: Document validation issues and corrections, track source verification, note manual overrides, maintain audit trails.

**Processing history**: Record AI decisions and rationale, document interventions, track agent performance, build institutional knowledge.

## Getting Started

1. Navigate to any dataset record
2. Click "Notebook" tab → "New Page"
3. Add title and emoji
4. Start writing with formatting and AI
5. Type `/ai` to generate content with record context
6. Edit and refine generated content

Record pages transform individual data points into rich, documented insights that grow more valuable over time through collaborative analysis.

## Properties View

### Field Data Display

The Properties tab presents your record data in a clean, searchable table format:

**Field Information**

- **Field Names**: Column names with data type indicators
- **Values**: Processed data with clear formatting based on field type
- **Evidence**: AI reasoning behind generated values (when applicable)
- **Sources**: Visual indicators showing original data sources with favicons

**Interactive Features**

- **Search Fields**: Filter visible fields by name or content
- **Value Formatting**: Automatic formatting for different data types
- **Source Browsing**: Click source indicators to see detailed attribution
- **Processing Status**: Visual indicators for AI processing states

### AI Evidence Tracking

For AI-processed fields, you can see:

- **Reasoning**: Detailed explanation of how the AI arrived at the result
- **Source Attribution**: Links to original web sources with visual previews
- **Confidence Indicators**: Understanding of result reliability
- **Processing History**: Timeline of when data was generated or updated

### Slash Commands

Type `/` to access a powerful command menu:

**Content Blocks**

- `/paragraph` - Plain text paragraphs
- `/heading1`, `/heading2`, `/heading3` - Section headings
- `/bulletlist` - Unordered lists
- `/orderedlist` - Numbered lists
- `/taskitem` - Checkable task items
- `/blockquote` - Quoted text blocks
- `/codeblock` - Code snippets with syntax highlighting
- `/table` - Interactive tables
- `/horizontalrule` - Section dividers

**AI Integration**

- `/ai` - Invoke AI assistant for content generation

### AI-Powered Content Generation

The notebook editor includes sophisticated AI integration:

#### AI Prompt Interface

When you select `/ai` or use the AI command:

**Smart Prompting**

- **Context Awareness**: AI understands the current record context
- **Agent Selection**: Choose from workspace AI agents
- **Real-time Streaming**: See content generate in real-time
- **Tool Integration**: AI can access external tools and data

**Content Generation**

- **Contextual Writing**: AI generates content relevant to the current record
- **Data Analysis**: Analyze field values and generate insights
- **Research Integration**: Pull in external information using available tools
- **Structured Output**: Generate tables, lists, and formatted content

#### AI Capabilities

**Content Types AI Can Generate**

- **Analysis Reports**: Detailed analysis of record data
- **Research Summaries**: External research about companies, contacts, or topics
- **Data Insights**: Patterns and observations from field values
- **Action Plans**: Next steps and recommendations
- **Comparison Tables**: Side-by-side comparisons with other records

**AI Tool Integration**

- **Web Search**: Access current information from the internet
- **API Calls**: Pull data from configured external APIs
- **Data Validation**: Verify information against multiple sources
- **Enrichment**: Add missing context and related information
